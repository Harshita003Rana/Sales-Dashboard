# Interactive Sales Dashboard in Microsoft Excel Using Pivot Tables and Pivot Charts
## Project Objective
The objective of this project is to create an interactive Sales Dashboard in Microsoft Excel that transforms raw sales data into meaningful insights. The dashboard helps analyze sales performance across cities, products, categories, sales representatives, and months, enabling better decision-making through data visualization and reporting.
## Dataset used
- <a href = "https://github.com/Harshita003Rana/Sales-Dashboard/blob/main/Sales%20Dashboard.xlsx">Dataset</a>
## Question(KPIs)
- What is the total sales revenue generated during the year?
- Which city contributed the highest sales revenue?
- Which product generated the maximum sales?
- Which product category contributed the most to total revenue?
- Who is the top-performing sales representative based on sales?
- Which month recorded the highest sales performance?
- How do sales trends vary across different months of the year?
